# JavaScript Mini Projects

A collection of beginner-friendly JavaScript mini projects built to practice DOM manipulation, event handling, and core JavaScript concepts.

##  Projects

### 1. Todo App
- Add new tasks
- View task list
- Built using JavaScript

### 2. Event Listener
- Demonstrates JavaScript event listeners
- Handles user interactions using click events

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

##  Folder Structure

```
javascript-mini-projects/
├── TodoApp/
├── Event-Listener/
└── README.md
```

##  How to Run

1. Clone the repository.
2. Open any project folder.
3. Open `index.html` in your browser.

This repository is created to improve JavaScript fundamentals through hands-on mini projects. More projects will be added as I continue learning.
